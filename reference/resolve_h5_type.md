# Resolves the HDF5 type based on the 'as' map and data properties

Resolves the HDF5 type based on the 'as' map and data properties

## Usage

``` r
resolve_h5_type(data, name, as_map)
```

## Arguments

- data:

  The R object.

- name:

  The name of the object (column name, dataset name, or attribute name).

- as_map:

  The processed 'as' argument (named vector).
